# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-28

### Added
- 初始版本发布
- 支持压缩和解压多种格式
- 自动识别模式
- 文件拖放支持
- 密码加密保护
- 内置 7z 二进制文件
- 自动生成输出路径
- 文件关联支持

### Features
- **压缩功能**：7z、zip、tar、gzip 格式
- **解压功能**：支持多种压缩格式
- **智能特性**：自动识别、自动路径
- **用户界面**：简洁易用的图形界面

### Bug Fixes
- 修复路径识别问题
- 优化错误处理
